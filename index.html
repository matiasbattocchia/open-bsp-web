What it is

OpenBSP connects your stack directly to Meta's WhatsApp Cloud API. No third-party BSP between you and Meta. Self-host on Supabase or run the hosted version — same code.

  • Self-hostableFork, set GitHub secrets, run the Release action. Live in under 15 minutes.
  • Multi-tenantRow-level security, role-based access. Run it for yourself or offer it to clients.
  • AI-agent readyDecoupled. Plug in any LLM framework via Chat Completions, A2A, or MCP. Built-in MCP server for agentic clients.
  • StackTypeScript, Deno Edge Functions, Postgres (Supabase). React + Tailwind on the UI.
  • LicenseUnlicense (public domain). Use it however you want.

Cost compared to a SaaS BSP

Supabase's free tier covers SMB volumes. Pro ($25/mo) kicks in at higher traffic. You can also self-host Supabase itself, but that's another story.

Web interface

WhatsApp Web-like UX. Conversations, contacts, templates, AI agents, stats. Responsive, dark mode, three languages.
